CSS(Cascade Style sheet)
->Style the page
->Its not programming language
Three types of writing CSS:
1.Inline CSS: write css with the html tag using style attribute
2.Internal CSS: write css inside style tag
3.External CSS: write css inside seperate file.And link using link tag

Property:It is the styling property such as border,color,font
Value :it is the value of property
Selector : Way of selecting tag

->Inline Structure
style="property:value;property:value"
->Internal CSS and External CSS Stucture:
selector {
    property:value;
    property:value;
}
CSS property list:
background-color -> set backbround color
color -> set text color
height -> set height of block
width -> set width of block
text-align -> set the alignment of text
border -> set the border
background-image -> set backround image
margin -> set margin 
padding -> set padding
